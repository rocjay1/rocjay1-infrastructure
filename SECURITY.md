| Version | Supported          |
| ------- | ------------------ |
| main    | :white_check_mark: |

## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please report it responsibly.

You can report vulnerabilities by:
- Opening a [draft security advisory](https://github.com/rocjay1/rocjay1-infrastructure/security/advisories/new) on GitHub.
- Contacting @rocjay1 directly via GitHub.

Please do not report security vulnerabilities through public GitHub issues.

We will acknowledge your report and provide an estimated timeframe for a fix.
